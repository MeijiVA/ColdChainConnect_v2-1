───────────────────────────────────────────────────────────────────────────────
  QUICK START (3 STEPS)
───────────────────────────────────────────────────────────────────────────────

1. INSTALL POSTGRESQL
   macOS:  brew install postgresql@15 && brew services start postgresql@15
   Linux:  sudo apt update && sudo apt install postgresql postgresql-contrib
   Win:    Download from postgresql.org

2. CREATE DATABASE
   psql -U postgres
   CREATE DATABASE coldchain_dev;
   CREATE USER coldchain_user WITH PASSWORD 'your_password';
   GRANT ALL PRIVILEGES ON DATABASE coldchain_dev TO coldchain_user;
   \q

3. INITIALIZE & RUN
   Update .env with DATABASE_URL
   pnpm db:push
   pnpm db:seed
   pnpm dev

Server: http://localhost:8080
